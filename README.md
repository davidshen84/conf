#gist

my personal gist :)

- Grab [highlight-indentation][1] and drop it at ~/.emacs.d
- Install the following from *Packages*
 - *js2-mode*
 - *org-mode*
 - *auto-complete*
 - *markdown-mode*

  [1]: https://raw.githubusercontent.com/antonj/Highlight-Indentation-for-Emacs/master/highlight-indentation.el
